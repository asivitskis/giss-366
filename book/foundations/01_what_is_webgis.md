## Learning Objectives

By the end of this unit, you will be able to:

- Describe what Web GIS is and how it differs from desktop GIS
- Explain the client-server model underlying every web map
- Identify the key differences between proprietary and open-source/cloud-native publishing landscapes
- Set up your course development environment (both stacks)
